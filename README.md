# VueJS, Quiz App using External API

## Brief
A VueJS Quiz SPA that uses an Quiz API to generate questions and answers.
The API Can be found here: https://opentdb.com/


## User Guide:
* Acess URL
* Attempt Quiz Questions
* See How Well you do!
